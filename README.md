# 启航志愿 · 四川省2026年高考志愿填报辅助系统

纯静态单页应用,零外部依赖。在线访问:https://skyhumarvel.github.io/gaokao-zhiyuan/

## 功能

- **智能冲稳保推荐** — 输入分数自动换算全省位次,按录取概率分「冲 / 稳 / 保」三档推荐院校
- **一分一段表** — 基于 2025 年四川省官方公开数据(本科线上区间),分数↔位次双向换算
- **院校库(84 所)** — 近 5 年(2021–2025)在川录取线、走势图、多维筛选
- **院校对比** — 最多 4 所横向对比近 5 年数据
- **志愿表** — 最多 45 个志愿,排序、导出 CSV、打印,本机保存
- **专业库(48 个)** — 按门类筛选,含就业方向与参考薪资
- **历年批次线** — 2021–2025 官方批次线一览
- 深色模式 / 移动端适配

## 数据说明

- 批次线与一分一段数据源自四川省教育考试院公开发布
- 各校历年录取线整理自省考试院调档线公告、高校招生网及公开教育媒体,部分为多方资料整理
- 2021–2024 为老高考文/理科口径,2025 起为新高考物理/历史类口径,跨口径分数不可直接比较

> ⚠ 数据可能存在口径差异(校区/专业组/批次)或误差,仅供参考,正式填报请以四川省教育考试院及各高校官方公布为准。
