# 启航志愿 · 四川省2026年高考志愿填报辅助系统(演示版)

纯静态单页应用,零外部依赖。在线访问:https://skyhumarvel.github.io/gaokao-zhiyuan/

## 功能

- **智能冲稳保推荐** — 输入分数自动换算全省位次,按录取概率分「冲 / 稳 / 保」三档推荐院校
- **一分一段表** — 分段查询 + 分数↔位次双向换算器(支持跨年等位分对照)
- **院校库(84 所)** — 搜索、省份/类型/985/211/双一流筛选、近三年投档线走势、详情弹窗
- **院校对比** — 最多 4 所院校横向对比
- **志愿表** — 最多 45 个志愿,排序、导出 CSV、打印,数据保存在本机浏览器
- **专业库(48 个)** — 按门类筛选,含就业方向与参考薪资
- **深色模式 / 移动端适配**

## 部署

仓库 Settings → Pages → Source 选 `Deploy from a branch`,Branch 选 `main` / `(root)`,保存即可。

> ⚠ 本站为技术演示项目,所有分数线、位次、招生计划等数据均为模拟示例,正式填报请以四川省教育考试院官方数据为准。
